# 🗺️ **FlightMap** (Vanilla 1.12.1)

Dhask's FlightMap for World of Warcraft 1.12.1 with the following enhancements/changes/additions:

 - Complete multi-hop flight data for all continents and both factions.
 - [**UnitXP_SP3**](https://codeberg.org/konaka/UnitXP_SP3/wiki) support. If the UnitXP_SP3 mod & its addon are installed, two new options will be available: "Notify: taskbar icon" & "Notify: system sound". When alt-tabbed, the first option will cause the WoW icon on the taskbar to flash when the flight path destination has been reached. The second option will play a system sound upon arriving when alt-tabbed. Both options can be enabled at the same time.
 - Option to hide/show flight master icons on the continent map.
 - Option to hide/show flight master tooltips. 
 - Option to hide/show the zone tooltip.
 - Option to hide/show level ranges on the zone tooltip.
 - Slider to increase/decrease the font size in the zone tooltip.
 - The addon now only writes flight paths with a difference of over 1 second to the savedvariable. Previously, every flight path would be written.
